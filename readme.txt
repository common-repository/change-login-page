Rebrand easily the login page of your WordPress installation.

== Description ==

Rebrand your WordPress Login page with some easy settings. With this plugin, it is possible to change the background image, change the default logo, change the text color and change the buttons.

Kindly let us know your feedback or comments to add more features to this plugin.

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Customize Login Page** in the search box and click on the search button.
3. Find Customize Login Page plugin.
4. Then click on Install Now after that activate the plugin.

== Frequently Asked Questions ==

= What can you change with the Customize Login Page? =
- Background image of the login Page
- Logo of the login Page
- Button background color and button text color
- Text color

= How to configure Customize Login Page? =

1. Click the "Settings" tab from the left navigation menu:
2. Click the "Login Logo" menu under the Settings menu
3. Upload new login logo using WP media uploader.
4. Click the "Save Changes" button to save the configuration details.
